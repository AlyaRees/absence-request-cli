## üì¶ Installation

Download and unzip the zip file in your chosen location on your local machine.

Open in an IDE (ideally Intellij or Visual Studio Code but the application should work in any IDE).

(Intellij and VSC) 
Navigate to your chosen IDE and open a new file.
Navigate to the Main.java file.
In the top right hand corner, click the 'Run' (Intellij) or 'Run Java' (Visual Studio) button.
You're all set!

## ‚ñ∂Ô∏è Usage

You should be greeted with the following menu.

Option 1 is to book holiday. 

There are 3 different types of absence you can log. Holiday, sickness or lateness to work.

You will need to provide some details (e.g. fullname, employee number and the amount of time you're logging etc).

Employee numbers should be exactly 6 digits long.

Type "y" for yes and "n" for no. This input is case insensitive. 


Select option 2 from the main menu to view all requests.

Option 3 allows you to login as an 'Admin' user and review all absence records. You can login by entering the password.

The password is:

(Shhh, don't tell anyone!)

You can APPROVE or DECLINE requests. 
All requests are PENDING APPROVAL until reviewed and can be changed by an admin user at any point.
